# Java Grade Calculator

My first Java project developed while learning Java.

## About

This application reads 4 student grades, calculates the final average and displays the student's academic status.

## Features

- Read 4 grades from the user
- Calculate the final average
- Display:
  - Approved
  - Recovery
  - Failed

## Technologies

- Java

## Concepts Practiced

- Scanner
- Variables
- User input
- Mathematical operations
- Conditional statements (if, else if, else)

## Example

Input:

A1: 7.5
A2: 6.0
A3: 8.0
A4: 7.0

Output:

Final Grade: 7.13
Approved
